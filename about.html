PROJECT INFO

// HOW TO CONTRIBUTE
  • 1. Fork the repository on GitHub
  • 2. Edit vendors.json, builders.json, or brands.json
  • 3. Follow the existing JSON structure exactly
  • 4. Submit a Pull Request with a clear description
  • 5. A maintainer will review and merge
// DATA STRUCTURE
  • → vendors.json — all 19+ vendor entries
  • → builders.json — DIY builder listings
  • → brands.json — official brand stores
  • → Each entry supports a warning flag + message
  • → Update shared.js to reflect JSON changes
// TECH STACK
  • ● HTML / CSS / Vanilla JavaScript — no frameworks
  • ● Data stored in shared.js (mirrors JSON files)
  • ● Hosted on GitHub Pages (free, static)
  • ● NeoBrutalist design system
  • ○ Zero dependencies, zero build steps
// GUIDELINES
  • ✓ Only list vendors actually operating in India
  • ✓ Use warning: true for community-reported issues
  • ✓ Keep descriptions factual and neutral
  • ✓ Verify URLs are live before submitting
  • ✗ No affiliate or paid placements
